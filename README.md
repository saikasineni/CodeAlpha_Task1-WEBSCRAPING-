Project Title: Web Scraping Book Dataset (500 Records)

This project demonstrates an end-to-end web scraping workflow using Python to automatically collect book data from the website Books to Scrape. The script extracts information such as book title, price, rating, and availability and converts the data into a structured dataset for analysis.

The project was developed using Jupyter Notebook and Python libraries including Requests, BeautifulSoup, and Pandas.

The scraper automatically navigates through multiple pages of the website and collects 500 book records, demonstrating how web data can be transformed into a structured dataset for data analytics and research.

Project Features

Automated multi-page web scraping

Extraction of structured product data

Dataset creation using Pandas

Exporting data into CSV format

Basic dataset preparation for analysis
